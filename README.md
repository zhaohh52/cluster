# 如何使用集群 

## 1. 账号申请 
联系管理员创建账号密码（如zhangsan/123456）
## 2. 远程客户端及传输软件安装
### 2.1 下载及安装
推荐Xshell和Xftp
下载地址：https://www.netsarang.com/zh/free-for-home-school/
### 2.2 连接
以Xshell为例，打开软件,点击左上角 ‘新建’，依次填写名称、主机及端口号

| 名称 | 主机 | 端口号 |
| ------ | ------ | ------ |
| 自定义 | 192.168.20.4 | 22 |

点击左上角的“用户身份验证”，依次填写管理员提供的用户名和密码，点击确定，连接成功。
